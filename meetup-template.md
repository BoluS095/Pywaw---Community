# PyWaw Meetup Template 📋

Standard template for organizing monthly PyWaw meetups.

## Pre-Meetup (2-3 weeks before)

### Speakers
- [ ] Confirm 2-3 speakers
- [ ] Collect talk titles and descriptions
- [ ] Share speaker guide with first-timers
- [ ] Confirm talk durations

### Logistics
- [ ] Book venue
- [ ] Confirm AV setup (projector, mic if needed)
- [ ] Arrange food/drinks sponsor (optional)
- [ ] Update Meetup.com event page

### Promotion
- [ ] Post on Meetup.com with full agenda
- [ ] Share on social media
- [ ] Send reminder to mailing list

## Meetup Day

### Agenda Template

```
18:00 - 18:15  Doors open, networking
18:15 - 18:20  Welcome & announcements
18:20 - 18:50  Talk #1 (30 min)
18:50 - 19:00  Q&A + break
19:00 - 19:30  Talk #2 (30 min)
19:30 - 19:40  Q&A + break
19:40 - 19:55  Lightning talk (15 min)
19:55 - 20:00  Closing remarks
20:00+         Networking / afterparty
```

### Host Checklist
- [ ] Arrive 30 min early for setup
- [ ] Test projector and WiFi
- [ ] Welcome attendees, point to food/drinks
- [ ] Introduce each speaker (name, topic, 1-sentence bio)
- [ ] Keep time — give 5-min and 1-min warnings
- [ ] Thank speakers and sponsors
- [ ] Announce next meetup date

## Post-Meetup

- [ ] Thank speakers (personal message)
- [ ] Upload slides/recordings if available
- [ ] Post photos on Meetup page
- [ ] Note attendance numbers
- [ ] Start planning next month

---

*Consistency is key. PyWaw has run monthly since 2016 because we follow this process every time.*
