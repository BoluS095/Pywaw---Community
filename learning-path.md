# Python Learning Path 🐍📚

Recommended resources for developers at every level. Curated by PyWaw organizers and community members.

## 🟢 Beginner (0-6 months)

### Fundamentals
1. **Python official tutorial** — [docs.python.org/3/tutorial](https://docs.python.org/3/tutorial/)
2. **Automate the Boring Stuff** — Al Sweigart (free online)
3. **Python Crash Course** — Eric Matthes

### Practice
- [Exercism Python Track](https://exercism.org/tracks/python) — Mentored exercises
- [Codewars](https://www.codewars.com/) — Kata-style challenges
- [Project Euler](https://projecteuler.net/) — Math + programming

### Key Topics
- Variables, types, operators
- Control flow (if/elif/else, loops)
- Functions and scope
- Lists, dicts, sets, tuples
- File I/O
- Basic error handling (try/except)

## 🟡 Intermediate (6-18 months)

### Books & Courses
1. **Fluent Python** — Luciano Ramalho
2. **Effective Python** — Brett Slatkin (90 specific tips)
3. **Python Cookbook** — David Beazley & Brian K. Jones

### Key Topics
- Object-oriented programming (classes, inheritance, protocols)
- Generators and iterators
- Decorators and context managers
- List/dict/set comprehensions
- Virtual environments and packaging
- Testing with pytest
- Type hints

### Frameworks to Explore
- **Django** or **FastAPI** — Web development
- **Pandas** — Data manipulation
- **Click** or **Typer** — CLI tools

## 🔴 Advanced (18+ months)

### Deep Dives
1. **CPython Internals** — Anthony Shaw
2. **Architecture Patterns with Python** — Harry Percival & Bob Gregory
3. **High Performance Python** — Micha Gorelick & Ian Ozsvald

### Key Topics
- Metaclasses and descriptors
- Async/await and asyncio
- C extensions and Cython
- Memory management and profiling
- Design patterns in Python
- Contributing to CPython
